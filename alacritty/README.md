# Installation

Add this to `~/.config/alacritty/alacritty.yml`

## Light

```yaml
colors:
  primary:
    background: "0xfafafa"
    foreground: "0x52565f"

  normal:
    black: "0x52565f"
    red: "0xd15f5f"
    green: "0x688d31"
    yellow: "0xc18401"
    blue: "0x5079be"
    magenta: "0xb05ccc"
    cyan: "0x338f87"
    white: "0x949ba5"

  bright:
    black: "0x52565f"
    red: "0xd15f5f"
    green: "0x688d31"
    yellow: "0xc18401"
    blue: "0x5079be"
    magenta: "0xb05ccc"
    cyan: "0x338f87"
    white: "0x949ba5"
```
